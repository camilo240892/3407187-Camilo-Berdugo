# Proyecto Semana 03: Calculadora de Dominio

🎯 **ÚNICO ENTREGABLE:** Este proyecto es el único entregable obligatorio para aprobar la semana.

---

## 🎯 Objetivo

Construir una calculadora multi-operación aplicada al dominio de **venta de productos agrícolas**, utilizando operadores aritméticos, de asignación compuesta, de comparación estricta y operadores lógicos para simular reglas reales del negocio.

---

## 📋 Tu Dominio Asignado

**Dominio:** Venta de productos agrícolas 🌽🍎

La calculadora simula una tienda agrícola donde se realizan cálculos como:

* Total de ventas por producto
* Ingreso total de la tienda
* Aplicación de descuentos
* Validación de tipo de cliente
* Evaluación de compras al por mayor

---

## ✅ Requisitos Funcionales (Cumplidos)

La calculadora incluye:

✔ **Operaciones aritméticas** (`+`, `-`, `*`, `/`, `%`, `**`)

* Cálculo de totales por producto
* Suma total de ventas
* Promedios
* Validación de números pares/impares

✔ **Asignación compuesta** (`+=`, `-=`, `*=`, `/=`)

* Acumulación de ingresos
* Aplicación de descuentos
* Ajustes de valores

✔ **Comparación estricta** (`===`, `!==`, `>=`, `>`)

* Validación de tipo de cliente
* Verificación de compra mínima
* Evaluación de cantidades

✔ **Operadores lógicos** (`&&`, `||`, `!`)

* Descuentos por cliente frecuente
* Compras al por mayor
* Combinación de condiciones del negocio

✔ **Numeric separators**

* Uso de valores como `50_000`, `2_000` para mejor legibilidad

✔ **Salida clara en consola**

* Uso de `console.log()` con mensajes descriptivos

## 🧠 Descripción general del sistema

Esta calculadora representa una tienda de productos agrícolas donde se procesan ventas de diferentes productos (manzanas, bananos y naranjas). El sistema calcula ingresos, aplica descuentos según condiciones específicas y evalúa comportamientos de compra, simulando una lógica básica de negocio en JavaScript.
